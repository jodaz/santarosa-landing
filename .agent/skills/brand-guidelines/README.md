# Santa Rosa Brand Guidelines Skill

This skill provides comprehensive brand guidelines for the Santa Rosa landing page project. It ensures that any new development adheres to the established visual identity, typography, and color palette.

## Installation

This skill is designed to be used with the Gemini CLI. To install it locally in your project:

1. Ensure you have the Gemini CLI installed.
2. Place this directory in your project's `.agent/skills/` folder.

## Usage

You can trigger this skill by asking the Gemini CLI about Santa Rosa's design or brand. For example:

- "Show me the Santa Rosa brand colors."
- "What fonts are used in Santa Rosa?"
- "How should I style a new card in Santa Rosa?"
- "What are the brand guidelines for this project?"

## Contents

- **Core Typography:** Poppins, Edo, and Permanent Marker.
- **Color Palette:** Detailed HSL and Hex values for brand colors.
- **Visual Language:** Gradients, shadows, and transitions.
- **Animations:** Standard animations used for interactive elements.
- **Design Principles:** Core rules for maintaining brand integrity.

## Maintaining the Skill

When the design system evolves (e.g., in `tailwind.config.ts` or `globals.css`), please update the `SKILL.md` file in this directory to reflect the changes.
